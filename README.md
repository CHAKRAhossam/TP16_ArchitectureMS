# TP16 - React GraphQL Client Application

## 📋 Description

Modern React frontend application using Apollo Client to consume GraphQL APIs. Features a responsive UI with Tailwind CSS for bank account management.

## 🛠️ Technologies

| Technology | Purpose |
|------------|--------|
| React 18 | Frontend Framework |
| Apollo Client | GraphQL Client |
| Tailwind CSS | Styling |
| TypeScript | Type Safety |

## 📁 Project Structure

```
TP16-Microservices/
├── src/
│   ├── apollo/
│   │   └── client.js          # Apollo configuration
│   ├── components/
│   │   ├── CompteList.js      # Account list component
│   │   ├── CreateCompte.js    # Account creation form
│   │   └── TransactionForm.js # Transaction handling
│   ├── graphql/
│   │   ├── queries.ts         # GraphQL queries
│   │   ├── mutations.ts       # GraphQL mutations
│   │   └── types.ts           # TypeScript types
│   └── App.js
├── package.json
└── README.md
```

## 🚀 Getting Started

```bash
git clone https://github.com/CHAKRAhossam/TP16_ArchitectureMS.git
cd TP16_ArchitectureMS
npm install
npm start
```

Open `http://localhost:3000` in your browser.
